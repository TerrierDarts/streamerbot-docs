Start an ad break on your Twitch channel

## Parameters
### `Duration`
Duration of ad break, in seconds.

- Type: `Integer`
- Values: `30`, `60`, `90`, `120`, `150`, `180`

::list{type=warning}
- Only the specified values are allowed
::
